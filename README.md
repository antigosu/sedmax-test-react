# Test homework for Sedmax

You can see a demo page at [**GitHub Pages**](https://antigosu.github.io/sedmax-test-react) :punch:

#### Installation

```
git clone git@github.com:antigosu/sedmax-test-react.git
```

```
cd sedmax-test-react
```

```
yarn install
```

#### Development

```
yarn start
```

#### Running the tests

```
yarn test
```

#### GitHub Pages deploy

```
yarn run deploy
```
